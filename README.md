# retail-business-analytics
Business analytics case study: sales, inventory and profit analysis using Power BI dashboards.
<h1 align="center">📊 Retail Business Analytics – Beverage Store Case Study</h1>

## Objetivo

Este projeto tem como objetivo aplicar **Business Intelligence e análise de dados** para melhorar a gestão de um pequeno negócio de varejo (depósito de bebidas), trazendo maior visibilidade sobre **vendas, margem de lucro e controle de estoque**.

Para isso, foram desenvolvidos:

* Um **processo de governança e padronização de dados** para garantir consistência nas entradas de vendas.
* Um **dashboard interativo no Power BI** para análise de desempenho do negócio.
* Estrutura de análise para identificar **produtos mais vendidos, margem de lucro e padrões de consumo**.

---

## Tecnologias Utilizadas

* **Power BI:** desenvolvimento de dashboards e análise de KPIs.
* **Excel:** estruturação e padronização da base de dados.
* **Modelagem de Dados:** organização das tabelas para análise de vendas e estoque.
* **Business Analytics:** análise de margem, ticket médio e giro de estoque.

## Insights Gerados

A análise permitiu identificar:

* produtos com **baixa margem de lucro**, sugerindo revisão de preço ou fornecedor
* produtos com **alto giro de vendas**, auxiliando decisões de reposição de estoque
* padrões de consumo por período
* oportunidades de **promoções e combos para aumentar vendas**

---

## Impacto no Negócio

* redução do tempo de fechamento de caixa
* maior controle sobre vendas e estoque
* visibilidade clara sobre a margem de lucro
* suporte à tomada de decisão baseada em dados
